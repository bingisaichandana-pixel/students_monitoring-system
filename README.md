# Student Monitoring System

## 📌 Description
This project is a web-based application to monitor student performance including marks, homework, participation, and conduct.

## 🛠 Technologies Used
- Python (Flask)
- MySQL
- HTML, CSS

## ✨ Features
- Student profile management
- Marks tracking
- Homework tracking
- Participation monitoring
- Reports generation

## ▶ How to Run
1. Install dependencies
2. Run app.py
3. Open browser at http://127.0.0.1:5000
